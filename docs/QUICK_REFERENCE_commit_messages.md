# Quick Reference: Finding GitHub Copilot Commit Messages

## TL;DR

**The commit message is stored IN the git commit itself, not in a separate file.**

## Quick Commands

### View the latest commit:
```bash
git log -1
```

### View a specific commit by hash:
```bash
git show <commit-hash>
```

### View full details (author + committer):
```bash
git log -1 --format=fuller <commit-hash>
```

### View just the message:
```bash
git log -1 --format=%B <commit-hash>
```

### Find all Copilot commits:
```bash
git log --committer="GitHub" --oneline
```

## Where is the "Commit Doc"?

There is no separate commit document file. The commit message is part of the Git commit object stored in `.git/objects/`. You access it using git commands like `git log` or `git show`.
